# 📊 Retail Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed using the Superstore dataset. It provides insights into sales performance, profitability, customer behavior, product performance, geographic trends, shipping analysis, returns analysis, and regional manager performance.

The dashboard helps business users monitor KPIs, identify trends, and make data-driven decisions through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze sales and profit performance
- Identify top-performing customers and products
- Evaluate regional sales performance
- Monitor shipping performance
- Analyze returned orders
- Evaluate regional manager performance
